LM3 算出式
LM3 = ( R1 + R2 + R3 ) ÷ R1
ROUND 1
収入（R1）
対象事業者の年間総収入
ROUND 2
地域内支出（R2）
仕入・人件費・委託費等の地域内支出
ROUND 3
地域内再支出（R3）
R2受取者による地域内での再支出
参考：全国平均 1.4〜1.7 ／ 優良事例 2.0以上
Problem & Solution

お金の「域外漏出」から
「域内循環」へ

大型店・ネット通販などの普及により、地域の事業者が稼いだお金が域外へ流出し、地域経済の空洞化が進んでいます。LM3はこの課題を数値で捉え、改善の道筋を示します。

✕ LM3スコアが低い状態　1.3以下（全国平均以下の経済循環率）
お金の大半が地域外へ流出
仕入・委託費・消費が域外の企業・サービスへ流れ、地域経済への波及効果が低下
地域経済の空洞化
地域内に残るお金が少なく、雇用・税収・地元産業が縮小するスパイラルに
○ LM3スコアが高い状態　2.0+（優良事例のスコア水準）
お金が地域内で何度も循環
地元の仕入先・スタッフ・業者を活用することで、お金が地域内で繰り返し使われる
地域経済の活性化
域内循環が高まることで、雇用・消費・税収が連鎖的に増加し、地域全体が潤う
Features

LM3メーターの4つの特徴

専門知識不要で、収入・支出・消費率を入力するだけでLM3を自動算出。調査から分析・比較まで一貫してサポートします。

FEATURE 01
自動でLM3を算出

収入・支出・消費率の数値を入力するだけで、LM3をリアルタイムで自動計算します。特に専門知識は必要ありません。

FEATURE 02
調査票が付属・配布も簡単

LM3の測定に必要な事業者・スタッフ・仕入先等の域内支出額データの収集を簡略化。関係者ごとに調査票Excelを自動生成し、メール添付で送るだけで回答収集を完了できます。

FEATURE 03
結果をExcelでダウンロード

アンケート結果・LM3算出過程・結果サマリーの3シートをExcelで出力可能。報告書や他資料への活用も簡単にできます。

FEATURE 04
データベースによる他事例との比較

当社が収集した事例のLM3スコア・各関係者ごとの域内調達率のデータから、当該事業の特性を分析。経済循環率を高めるための手法の検討にも活用可能です。
